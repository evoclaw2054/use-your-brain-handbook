# Schema (心智框架)：你的知識變形金剛

## 什麼是 Schema？

想像你的大腦是一個倉庫，知識是一個個包裹。如果每個包裹都要你手動管理，那會太累。
所以你的大腦會建立 **Schema（知識結構／心智框架）**。你可以把它想成倉庫的**大架子**，架子可以放很多包裹。

> **Schema 不是固定的。它會慢慢變，像變形金剛一樣重組、調整形狀，去配合你正在解的問題。**

這是人腦最強大的地方：
*   人腦不只是存資訊。
*   人腦會**操作結構**。
*   人腦會**重排意義**。
*   人腦會**建立關係**。
*   人腦會把內部模型調到符合當下情境。

## 為什麼 AI 無法取代你建立 Schema？

NotebookLM 和 GPT 可以幫你摘要，也可以丟問題。這很有用。
但它們**無法用你的方式理解你的生活脈絡**，也**無法替你建立「你的內部 Schema」**。

*   **編碼是個人的**。
*   **結構也是個人的**。

只有你自己知道這些知識要放在你人生的哪個架子上，跟你的哪些經驗相連。

## 三層來回跳：建立模型的關鍵

新手最常犯的錯，是把建立 Schema 當成一次做完的流程。
其實，這是一個 **遞迴迴圈**。你要在三個層次之間**反覆跳躍**：

### 1. 主題層 (Topic Layer)
*   **內容**：這個概念是什麼？術語是什麼？它在主張什麼？
*   **功能**：確認基本事實與定義。

### 2. 邏輯層 (Logic Layer)
*   **內容**：什麼連到什麼？為什麼？機制是什麼？權衡是什麼？限制是什麼？
*   **功能**：建立因果、流程與關聯性。

### 3. 底層 (Bottom Layer)
*   **內容**：跟我有什麼關係？我要用在哪？我要做什麼？
*   **功能**：將知識轉化為個人行動與應用。

### 如何操作「來回跳」？

你不是只跑一次「主題 → 邏輯 → 底層」。你會不斷地：
*   **回主題**：因為你發現漏了東西。
*   **回邏輯**：因為你一想應用就發現缺口。
*   **回底層**：因為你要讓它變成行動。

**上下移動本身，就是流程。**

## 跨過門檻：從混亂到清晰

如果你剛開始做非線性筆記，先知道一件事：**你會有感覺，而且卡住很正常。**

*   **Level 1** 很像抄寫，很乾淨、很安全，但之後找不到。
*   **Level 2** 會開始有連結，偶爾有頓悟，但結構不穩。
*   **Level 3** 是整合，你會有更多燈泡瞬間，因為每個連結都會幫你看到下一個連結。

但它也更亂，更像跳著學。你會覺得「我現在到底在幹嘛」。
這就是 **門檻概念 (Threshold Concept)**。你一跨過去，就看得到結構，你就回不去了。洞穴還在，但電路升級了。
