# 02. Schema (心智框架)：理解的捷徑與陷阱

> 「我們看不見世界原本的樣子，我們看見的是我們『預期』的樣子。」

## 什麼是 Schema (心智框架)？

Schema 是大腦中既有的**知識結構**或**認知框架**。它像是一個心智模版，幫助我們快速理解新資訊。
* 當你看到「餐廳」，大腦自動啟動「點餐、吃飯、付帳」的 Schema。
* 當你讀到「AI」，大腦可能啟動「機器人、未來、威脅」或「工具、助手、效率」的 Schema。

**功能**：
1. **加速理解**：不用從零開始分析，直接套用既有框架。
2. **填補空白**：當資訊不足時，Schema 會自動補全缺失的部分（有時會出錯）。
3. **過濾雜訊**：自動忽略與框架無關的細節。

## 雙面刃：捷徑也是陷阱

### 陷阱一：確認偏誤 (Confirmation Bias)
我們傾向於尋找和解釋那些**符合**既有 Schema 的資訊，而忽略或排斥**不符合**的資訊。
* **例子**：如果你認為「某種投資一定賺」，你會只關注好消息，無視風險警告。

### 陷阱二：刻板印象 (Stereotype)
將對群體的過度簡化框架套用到個人身上，導致誤判。

### 陷阱三：功能固著 (Functional Fixedness)
只看到物體的傳統用途，看不到新可能。
* **例子**：只把迴紋針看作夾紙工具，想不到可以拿來當導線或開卡扣。

## 如何識別並打破舊框架？

要學習新東西，往往需要**摧毀**或**重構**舊的 Schema。

### 徵兆：你被舊框架卡住了嗎？
* 覺得新資訊「很怪」、「不合理」、「不可能」。
* 只能用舊有的詞彙去強行解釋新現象。
* 對反駁自己觀點的證據感到憤怒或排斥。

### 框架重構三步法 (Schema Reconstruction)

#### 步驟 1：覺察 (Awareness)
* **暫停**：當感到困惑或排斥時，停下來。
* **標記**：問自己：「我現在是用什麼框架在看這件事？」、「我的哪個假設被挑戰了？」

#### 步驟 2：解構 (Deconstruction)
* **拆解**：將既有框架拆成最小單位。
* **質疑**：逐一檢查這些單位的真實性。「這一定是真的嗎？」、「有沒有反例？」

#### 步驟 3：重組 (Reconstruction)
* **引入新視角**：主動尋找與舊框架不同的觀點。
* **建立新連結**：將新資訊與舊知識的**不同部分**連結，形成新結構。
* **比喻置換**：用一個全新的比喻來描述事物。
    * 舊：「大腦像電腦硬碟」（暗示容量有限、被動儲存）。
    * 新：「大腦像肌肉」（暗示越練越強、主動重組）。

## 實戰練習：框架置換遊戲

找一個你熟悉的觀念，強迫自己用三個完全不同的比喻來描述它。
* **主題**：學習
    * 比喻 A：學習像「蓋房子」（需要地基、結構）。
    * 比喻 B：學習像「種樹」（需要時間、耐心、自然生長）。
    * 比喻 C：學習像「編織網」（節點連結、越補越大）。

**反思**：這三個比喻分別強調了學習的哪些面向？忽略了哪些面向？

## 小結

Schema 是雙刃劍。善用它能事半功倍，依賴它則會固步自封。保持**認知彈性**，隨時準備好「重構」自己的心智框架，是深度學習的關鍵。
