# 02. Schema 升級：重構你的認知框架

## 2.1 什麼是 Schema（心智框架）？

Schema 是大腦中的「檔案櫃」或「劇本」。我們用既有框架去理解新事物。
* **正面**：快速理解（看到四條腿+汪汪叫 -> 狗）。
* **負面**：刻板印象或錯誤歸因（認為「資質差」所以「學不會」）。

## 2.2 識別錯誤框架

當你發現自己常有這些念頭，代表 Schema 可能錯了：
* 「我就是沒這方面的天份。」（固定型思維）
* 「這太難了，我做不到。」（能力邊界錯誤設定）

## 2.3 重構技術：認知衝突法

主動製造「預期」與「現實」的落差，強迫大腦更新框架。

### 操作步驟：
1. **寫下預期**：在做一件事前，先寫下你預測的結果（例如：這次簡報我會很緊張，效果會很差）。
2. **執行並記錄**：實際去做，並記錄真實反饋（例如：簡報中有一個笑點大家笑了，主管點頭）。
3. **對比分析**：找出預期與現實的差距，問自己：「為什麼我的舊框架預測錯誤？」
4. **更新腳本**：根據新證據，改寫大腦中的劇本（例如：原來準備充分就不會那麼糟）。

## 2.4 跨領域類比

將陌生領域映射到熟悉領域。
* **範例**：將「寫程式」類比為「烹飪」。
    * 變數 = 食材
    * 函式 = 食譜
    * 編譯錯誤 = 菜煮焦了

## 2.5 今日練習

找出一個你認為自己「天生不擅長」的領域，寫下你的舊框架，然後找出一個反例，強迫自己寫出一個新的解釋框架。
